# PRODIGY_WD_03

A web-based Tic Tac Toe game built with HTML, CSS, and JavaScript featuring AI gameplay with multiple difficulty levels and local two-player mode.

## Features
• AI opponent with Easy, Medium, and Hard difficulty  
• Unbeatable AI using the Minimax algorithm  
• Local Player vs Player mode  
• Interactive UI with restart option  

## Technologies Used
• HTML  
• CSS  
• JavaScript
